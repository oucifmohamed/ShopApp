## What I Used ?

- Clean Architecture : Basically I have devided the project into three layers (Data,Domain and presentation layer).
- Jetpack Compose.
- Retrofit.
- Kotlin Coroutines.
- Firebase Auth.
- Firebase Firestore.
- Firebase Push Notification.
- Kotlin Coroutines.
- MVVM pattern in the presentation layer.
- Unit Testing.
- Dagger Hilt.

## Run the app

To run the app clone the repository,open it on android studio add api key for google maps api and web api key to be able to implement firebase push notifications.
After that just run the app.
